# TypeScript Template Project

## 開発開始

次の yarn コマンドで必要なモジュールをインストールします。

```
$ yarn install
```

次のコマンドで実行します。

```
$ yarn start
```

ブラウザで「http://localhost:4000/ 」を開きます。
ソースコードを更新すると自動的にビルドされますので、「http://localhost:4000」をリロードして更新します。

test ディレクトリにテストコードを書いて次のコマンドでユニットテストを行います。

```
$ yarn test
```
